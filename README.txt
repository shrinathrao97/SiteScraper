

To run this program, in a terminal first ensure pip and python are installed: 
* `pip install`

If you have output, then move on, else run: 
* `sudo apt-get install pip`

Run:
* `python -m venv SiteScraperPythonVersion`
* `source <pathToSiteScraperPythonVersion>/bin/activate`

Now run: 
* `cd <repoDir>`
* `pip install -r requirements.txt`
* python siteScraper.py
